Langages De Programmation

Top 10 langages essentiels et leur histoire:

1. Le Code Machine

  • La base absolue. Tout le reste compile vers ça. Composé de zéros et de uns (ou hexadécimal pour nous).
  • Histoire: utilisé depuis les premiers ordinateurs des années 40, chaque CPU a son propre jeu d’instructions.
  • Exemple x86:
0xb0 0x1e 0xb3 0x03 0x00 0xd8

2. Assembly

  • Traduction lisible du code machine. Permet de contrôler directement le matériel.
  • Histoire: apparu dans les années 50; indispensable pour OS, BIOS, pilotes.
  • Exemple NASM x86-32:
mov al, 30
mov bl, 3
add al, bl
call $-1

3. Fortran (1957)

  • Premier vrai langage haut-niveau pour calcul scientifique et ingénierie.
  • Histoire: utilisé pour des calculs complexes sur des superordinateurs dès 1957.
  • Exemple:
WRITE (6,7)
7 FORMAT(13H HELLO, WORLD)
STOP
END

4. C (1972)

  • Langage système polyvalent. Base des OS modernes, compilateurs et logiciels critiques.
  • Histoire: créé pour réécrire Unix, influence directe sur C++, Objective-C, Rust, Go.
  • Exemple:
#include <stdio.h>
int main() {
  printf("Bonjour, C!\n");
  return 0;
}

5. C++ (1983)

  • Extension de C avec la programmation orientée objet.
  • Histoire: utilisé pour logiciels complexes, jeux, moteurs graphiques.
  • Exemple:
#include <iostream>
using namespace std;
int main() {
  cout << "Bonjour, C++!" << endl;
  return 0;
}

6. Python (1991)

  • Langage dynamique, simple, très lisible.
  • Histoire: Guido van Rossum le crée pour remplacer ABC; explosera avec scripting, web, IA.
  • Exemple Python 2:
name = raw_input("Entrez votre nom: ")
print "Bonjour,", name

7. Java (1995)

  • “Write once, run anywhere”; orienté objet et portable.
  • Histoire: inventé par Sun Microsystems, énorme pour entreprise et Android.
  • Exemple:
public class Hello {
  public static void main(String[] args) {
    System.out.println("Bonjour, Java!");
  }
}

8. JavaScript (1995)

  • Langage indispensable du web; Client et Serveurs.
  • Histoire: créé en 10 jours pour Netscape; devient moteur de tout navigateur moderne.
  • Exemple:
console.log("Bonjour, JavaScript!");

9. Rust (2010)

  • Langage moderne; sécurisé et performant.
  • Histoire: créé par Mozilla pour remplacer C/C++ là où sécurité mémoire et performance comptent.
  • Exemple:
fn main() {
  println!("Bonjour, Rust!");
}

10. C# (2000)

  • Langage par Microsoft, orienté objet, utilisé pour applications desktop, web, et jeux (Unity Principalement).
  • Histoire: conçu pour rivaliser avec Java, rapidement adopté dans l’écosystème Windows.
  • Exemple:
using System;
class Program {
  static void Main() {
    Console.WriteLine("Bonjour, C#!");
  }
}
